QUICK CRM — FULL ORIGINAL UI + REAL MILESWEB BACKEND

This build starts from the user's original full-feature Android project.
It preserves the original UI/screens and adds cloud authentication and lead sync via:
https://crm.rscc.in/api/index.php

Preserved modules include:
- Dashboard charts and metrics
- Lead list, detail, search, filters
- Pipeline Kanban
- Telecaller dashboard and access controls
- Auto dialer and call workflow UI
- Call history, annotations and playback UI
- WhatsApp screens, templates and chat workflow UI
- Facebook Lead Ads screens, diagnostics and polling service
- CSV import/export
- Reminder rules and follow-up dashboard
- Admin reports and PDF/CSV reporting logic
- Gemini sentiment/script features (requires a real key configured at build/runtime)
- Sync hub and logs

Real cloud-backed functions:
- Admin login/token session
- Lead download from MySQL
- Lead creation/update/delete
- Status/follow-up sync
- Telecaller/staff snapshot sync
- Facebook webhook leads through MilesWeb

External credentials must stay on the server. Do not put MySQL passwords in the APK.
