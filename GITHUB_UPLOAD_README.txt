QUICK CRM — COMPLETE ANDROID STUDIO PROJECT

Upload the CONTENTS of this folder to the root of your GitHub repository.
Do not upload the outer ZIP as the project source.

Required wrapper files included:
- gradlew
- gradlew.bat
- gradle/wrapper/gradle-wrapper.jar
- gradle/wrapper/gradle-wrapper.properties

Android Studio:
1. Clone the GitHub repository.
2. Open the repository root.
3. Use JDK 17 / Android Studio Embedded JDK.
4. Allow Gradle sync. The wrapper downloads Gradle 8.7 on first use.

Do not commit:
- local.properties
- *.jks / *.keystore
- keystore.properties
- *.apk / *.aab
- passwords or API secrets
